# Responsive Portfolio Website

A modern and fully responsive personal portfolio website developed using HTML5, CSS3, JavaScript, and jQuery.

## Features
- Responsive design for mobile, tablet, and desktop
- Modern dark-themed UI with smooth animations
- Skills, projects, certifications, and contact sections
- Interactive navigation menu
- Clean and user-friendly interface
- GitHub Pages deployment support

## Technologies Used
- HTML5
- CSS3
- JavaScript
- jQuery

## Project Highlights
- Designed using Flexbox and CSS Grid layouts
- Implemented responsive web design principles
- Applied modern UI/UX styling and hover effects
- Built recruiter-friendly professional portfolio layout
- Optimized for accessibility and smooth user experience
